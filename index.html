The reason I love design and more specifically UX design is because I really enjoy helping others. UX design, to me, is about creating an environment in which the user can not only succeed, but thrive. My goal in my design work is to design an experience for the user that is clear and simple to use. This means my job is to listen to my user's feedback and to be willing to adapt my project in any way that helps accomplish that task.
